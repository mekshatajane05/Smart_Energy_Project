# Smart Energy Consumption Analysis and Prediction using Machine Learning

## Project Overview

This project analyzes household energy consumption data and predicts future energy usage using Machine Learning and Deep Learning models.

## Objectives

* Analyze device-level energy consumption
* Perform data preprocessing and feature engineering
* Develop Linear Regression baseline model
* Implement LSTM forecasting model
* Generate smart energy-saving suggestions
* Build an interactive dashboard using Flask

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* TensorFlow/Keras
* Flask
* Matplotlib
* HTML/CSS/JavaScript

## Dataset

The project uses the Individual Household Electric Power Consumption dataset.
Due to GitHub file size limitations, the dataset is not included in this repository.
Download:https://archive.ics.uci.edu/ml/datasets/individual+household+electric+power+consumption
Place the file in:dataset/household_power_consumption.txt
